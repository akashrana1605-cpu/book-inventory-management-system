# 📚 Book Inventory Management System

A simple Book Inventory Management System built using **Node.js, Express.js, EJS**, and the **MVC Architecture**.

## 🚀 Features

- 🔐 User Authentication (Register/Login)
- 📚 Add, Update & Delete Books
- 🖼️ Image Upload with Multer
- 🌙 Dark & Light Theme
- 🍪 Session & Cookie Management
- ✅ Form Validation
- 🛡️ Protected Routes (Authentication)

## 🛠️ Tech Stack

- Node.js
- Express.js
- EJS
- HTML
- CSS
- Bootstrap
- Multer
- Express Session
- Cookie Parser

## ▶️ Run Locally

```bash
git clone https://github.com/akashrana1605-cpu/book-inventory-management-system.git
cd book-inventory-management-system
npm install
node index.js
```

Open: **http://localhost:3300**
